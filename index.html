사이트 매뉴얼
  1. CRM에서 차트 이미지 저장
    1. CRM에서 펜차트 > 상단 메뉴 중 [선택] 버튼 클릭
    2. 텍스트 추출할 차트를 오래된 순서대로 2개 선택 후 [비교] 버튼 클릭
    3. 2장의 차트 비교 화면이 뜨면 화면 캡쳐해서 저장
    4. 참고: 3장부터는 이미지 속 텍스트 선명도 떨어져서 인식률 저하되므로 최대 2장까지만 할 것!
  2. 차트 텍스트 추출
    1. 본 사이트의 [차트 이미지 첨부] 버튼 클릭 → 저장한 차트 이미지 불러오기
    2. [텍스트 추출] 버튼 클릭
  3. 텍스트 정리 & 복사
    1. [텍스트 정리] 버튼 클릭 *중요!*
    2. [정리결과 복사] 버튼 클릭해 메모장에 텍스트 붙여넣기
    3. *가장 중요!*실제 펜차트와 대조해 최종 확인 및 편집하기
    4. 특히 취소선 있는 텍스트는 인식 실패 확률 높으므로 꼭 체크!
